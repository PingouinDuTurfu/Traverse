# Traverse

Traverse is a board game for 2, 3, or 4 players, entirely developed in C.

## Libraries needed to be added.

- SDL 1: `libsdl1.2-dev`
- SDL 1 image: `libsdl-image1.2 libsdl-image1.2-dev`
- GLUT: `freeglut3-dev`

Compilation line
`gcc *c -lSDL -lSDL_image -lGL -lGLU -lglut -lm`
